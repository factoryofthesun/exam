EXAM: Experiment-As-Market
============================================================

Overview
~~~~~~~~

The exam package is a generic implementation of the experimental design and estimation method proposed in :cite:`Narita2020`. The package provides functions for computing welfare-optimal assignment probabilities, treatment assignment, and treatment effect estimation
